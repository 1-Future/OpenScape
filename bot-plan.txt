BOT IN GAME — PLAN
==================

GOAL
----
Claude joins miniscape as a player and chats naturally.

FLOW
----
1. Player types in game chat
2. Game server POSTs message to Discord webhook
3. Claude sees it in #game-bridge, responds
4. WS client watching Discord picks up Claude's reply
5. WS client sends reply back into game as "Claude: <msg>"

PIECES
------
[ ] Game server  — on chat message, POST to webhook URL below
[ ] WS client    — connects to Discord gateway, watches channel, pipes reply to game
[ ] Game server  — receives reply, broadcasts as "Claude: <msg>"

WEBHOOK POST (game → Discord)
------------------------------
POST to webhook URL with body:
  { "username": "William (in-game)", "content": "<player message>" }

WS CLIENT (Discord → game)
---------------------------
Connect to: wss://gateway.discord.gg/?v=10&encoding=json
Identify with bot token
Listen for MESSAGE_CREATE events in channel 1480654372131180635
Filter: ignore messages from bot user ID 1464768627709313044 (avoid loops)
On message → send to game server
